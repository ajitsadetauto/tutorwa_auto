    Open VS Code → File > Open Folder... → select C:\project file
    Install the Python extension (Microsoft).

7 - Configure pytest in VS Code
Create .vscode/settings.json:

{
  "python.testing.pytestEnabled": true,
  "python.testing.unittestEnabled": false,
  "python.testing.pytestArgs": [
    "-q"
  ]
}

8 - Pytest config
    Create pytest.ini in the project root:

        [pytest]
        addopts = -q -s
        testpaths = tests
        python_files = test_*.py *_test.py
        asyncio_mode = strict

9 - Create the tests folder:

        mkdir tests
